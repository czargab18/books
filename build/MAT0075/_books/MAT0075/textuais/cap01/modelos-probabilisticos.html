1 Alguns Modelos Probabilísticos

1.1 Introdução

Neste capítulo, introduzimos os conceitos fundamentais de probabilidade através de exemplos concretos e modelos básicos.

1.2 Espaço Amostral

O espaço amostral \(\Omega\) é o conjunto de todos os resultados possíveis de um experimento aleatório.

1.2.1 Definição 1.1

Dado um experimento aleatório, o espaço amostral \(\Omega\) é o conjunto que contém todos os possíveis resultados do experimento.

1.2.2 Exemplos

Exemplo 1.1 (Lançamento de uma moeda) - Experimento: lançar uma moeda - Espaço amostral: \(\Omega = \{C, K\}\) onde \(C\) = cara e \(K\) = coroa

Exemplo 1.2 (Lançamento de um dado) - Experimento: lançar um dado de 6 faces - Espaço amostral: \(\Omega = \{1, 2, 3, 4, 5, 6\}\)

Exemplo 1.3 (Tempo de vida de um componente) - Experimento: medir o tempo até a falha de um componente - Espaço amostral: \(\Omega = [0, +\infty)\)

1.3 Eventos Aleatórios

1.3.1 Definição 1.2

Um evento é um subconjunto do espaço amostral. Denotamos eventos por letras maiúsculas como \(A\), \(B\), \(C\), etc.

1.3.2 Operações com Eventos

Para eventos \(A\) e \(B\): - União: \(A \cup B\) (ocorre \(A\) ou \(B\) ou ambos) - Interseção: \(A \cap B\) (ocorrem \(A\) e \(B\) simultaneamente) - Complemento: \(A^c\) (não ocorre \(A\)) - Diferença: \(A \setminus B = A \cap B^c\) (ocorre \(A\) mas não \(B\))

1.3.3 Propriedades

  • \(A \cup A^c = \Omega\)
  • \(A \cap A^c = \emptyset\)
  • \((A^c)^c = A\)
  • \((A \cup B)^c = A^c \cap B^c\) (Lei de De Morgan)
  • \((A \cap B)^c = A^c \cup B^c\) (Lei de De Morgan)

1.4 Medida de Probabilidade

1.4.1 Definição Intuitiva

A probabilidade é uma função que atribui a cada evento um número real entre 0 e 1, representando a “chance” ou “verossimilhança” de ocorrência do evento.

1.4.2 Propriedades Básicas

Para qualquer evento \(A\): - \(0 \leq P(A) \leq 1\) - \(P(\Omega) = 1\) - \(P(\emptyset) = 0\) - \(P(A^c) = 1 - P(A)\)

1.4.3 Exemplo: Equiprobabilidade

Quando todos os resultados elementares são igualmente prováveis:

\[P(A) = \frac{\text{número de casos favoráveis}}{\text{número de casos possíveis}}\]
